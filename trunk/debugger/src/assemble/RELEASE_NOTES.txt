Release Notes For ehcache-debugger
==================================

This version of the debugger works with Ehcache 1.7.0.

See http://ehcache.sourceforge.net/documentation/remotedebugger.html for full documentation including installation and configuration.

For the impatient:

java -classpath slf4j-api-1.5.8.jar:slf4j-jdk14-1.5.8.jar -jar ehcache-debugger-1.7.1.jar ehcache.xml sampleCache1
